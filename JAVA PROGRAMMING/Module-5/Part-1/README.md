# Module 5 Part 1: Java Database Connectivity (JDBC)

This folder contains the practical implementation of direct database access using the JDBC API.

## 📝 Topics and Materials

| Topic             | Description                                 | Link                                     |
| :---------------- | :------------------------------------------ | :--------------------------------------- |
