# Module 3 Exam Questions: Exceptions, Threads & Packages

Review your knowledge of Java's powerful Exception Handling and Multithreading features.

## Section 1: Multiple Choice Questions (MCQs)

1. **Which keyword is used to handle exceptions in Java?**
   - A) check
   - B) try
   - C) handle
   - D) throw

2. **Which method is used to start a thread?**
   - A) run()
   - B) init()
   - C) start()
   - D) execute()

3. **What does the 'synchronized' keyword provide?**
   - A) Fast execution
   - B) Memory optimization
   - C) Thread safety (Mutually exclusive access)
   - D) Automatic Garbage Collection

4. **Which block always executes, regardless of whether an exception occurs?**
   - A) try
   - B) catch
   - C) finally
   - D) throw

5. **A package is declared using which keyword?**
   - A) folder
   - B) library
   - C) namespace
   - D) package

---

## Section 2: Coding Challenges

### Challenge 1: Exception Handling

Write a program that takes two integers from the user and performs division. Use `try-catch` to handle any `ArithmeticException` (division by zero).

### Challenge 2: Multithreading

Create two threads: One thread should print "Ping" and the other should print "Pong" every 500ms for 5 iterations. Use `Thread.sleep()` to manage timing.

---

## Answer Key

**MCQs:**

1. B (try)
2. C (start)
3. C (Thread safety)
4. C (finally)
5. D (package)
