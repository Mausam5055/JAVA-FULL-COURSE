# Module 2 Exam Questions: Java Object-Oriented Programming

Test your knowledge of Java OOP concepts with these practice questions!

## Section 1: Multiple Choice Questions (MCQs)

1. **Which keyword is used to inherit a class in Java?**
   - A) implements
   - B) inherits
   - C) extends
   - D) super

2. **What is the purpose of the 'this' keyword?**
   - A) To create a new object
   - B) To refer to the current object
   - C) To refer to the parent class
   - D) To stop method execution

3. **Which of these cannot be used for a constructor?**
   - A) public
   - B) static
   - C) protected
   - D) private

4. **What is encapsulation in Java?**
   - A) Performing multiple tasks at once
   - B) Wrapping data and code into a single unit
   - C) Inheriting methods from another class
   - D) Overloading methods

5. **Which keyword is used to prevent a class from being inherited?**
   - A) static
   - B) final
   - C) abstract
   - D) void

## Section 2: Coding Challenges

### Challenge 1: Inheritance

Create a parent class `Employee` with a method `work()`. Create a child class `Developer` that overrides the `work()` method to print "Developer is coding...".

### Challenge 2: Encapsulation

Create a class `Account` with a private variable `balance`. Provide public methods `deposit(double amount)` and `getBalance()`. Ensure that balance cannot be negative.

---

## Answer Key

**MCQs:**

1. C (extends)
2. B (Current object)
3. B (Constructors cannot be static)
4. B (Wrapping data and code)
5. B (final)
