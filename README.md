## üöÄ Getting Started

### Prerequisites

- **Java Development Kit (JDK)** installed (JDK 8 or higher recommended).
- A code editor or IDE like **IntelliJ IDEA**, **Eclipse**, or **VS Code**.

### Running the Code

1.  Clone the repository:
    ```bash
    git clone https://github.com/Mausam5055/JAVA-FULL-COURSE.git
    ```
2.  Navigate to the directory:
    ```bash
    cd JAVA-FULL-COURSE
    ```
3.  Open any file in your IDE or compile via terminal:
    ```bash
    javac YourFile.java
    java YourFile
    ```
